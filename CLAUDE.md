# CLAUDE

# Project Context

This is a high-end digital marketing agency website (SicaCreative).

The goal is NOT to look like a typical agency.
The goal is to feel like a **premium SaaS product + growth system**.

The website must communicate:
- Predictability
- Performance
- Measurable business outcomes

Target:
Clinics and local businesses ready to invest in growth.

---

# Design Philosophy

- Clean is not enough → it must feel **engineered**
- Every element must communicate value
- Avoid “template feeling” at all costs
- Prioritize **clarity, contrast, and hierarchy**
- Design should feel like a product, not a service

---

# Visual Identity

- Primary: #34224D (deep purple)
- Secondary: #6F5AA3
- Accent: #c6db5b (lime green for CTAs)

### Background Rules (CRITICAL)
- Use near-white backgrounds (#FBFBFD or #FFFFFF)
- Avoid dirty/washed gradients
- Gradients ONLY as subtle glow (never full background)
- Glow should be localized (e.g. behind dashboard visuals)

### Depth & Effects
- Soft shadows (not heavy)
- Subtle glow on key elements (CTA, active UI)
- Optional light glassmorphism (very subtle)

---

# Typography

- Modern sans-serif (Inter style)
- Strong, bold headlines
- Slightly increased line-height for large headings
- Limit paragraph width (max ~520px)
- Body text must be lighter than headings (visual contrast)

---

# Layout Rules

- Max width: 1200px
- Generous vertical spacing (premium feel)
- Avoid compressed layouts
- Prefer 2-column hero sections
- Strong alignment and spacing consistency

---

# UX & Conversion Principles

- Above the fold must:
  → Hook attention (pain or tension)
  → Present outcome
  → Show system (visual proof)

- Flow:
  Problem → Mechanism → Proof → CTA

- Always answer:
  → Why should I trust this?
  → How does this work?
  → What result do I get?

---

# Hero Section Rules (VERY IMPORTANT)

Hero must include:

1. **Emotional hook (small badge above title)**
   Example:
   - “Muito trabalho. Pouco crescimento.”
   - “Agenda cheia, faturação aquém?”

2. **Strong outcome-driven headline**
   - Short
   - Memorable
   - No generic agency wording

3. **Clear subtext**
   - Explain mechanism simply
   - Focus on predictability

4. **Primary + Secondary CTA**

5. **Trust indicators (badges or metrics)**

6. **Right-side visual MUST feel like a real system**
   - SaaS-style dashboard
   - Metrics (growth, CAC, ROAS)
   - Clean UI cards

---

# Motion & Micro-Interactions (DIFFERENTIATOR)

Use motion to increase perceived value:

- Chart line should animate (draw effect)
- Numbers should count up
- Status indicators (e.g. “Atualizado agora”) should pulse
- Buttons:
  → hover scale + shadow
- Secondary CTA:
  → arrow movement or underline animation

Optional:
- Left vertical scroll progress bar (thin, gradient)

Avoid heavy or distracting animations.

---

# CTA Rules

- Primary CTA = lime green
- Must stand out immediately
- Hover = scale + stronger shadow

- CTA copy:
  → action-driven
  → specific
  → outcome-oriented

Avoid generic:
- “Saiba mais”
- “Contacte-nos”

---

# Copywriting Rules

- European Portuguese
- Clear, direct, confident

### Must:
- Focus on outcomes (revenue, leads, predictability)
- Use tension (problem vs result)
- Be concise

### Avoid:
- Generic agency language
- Buzzwords
- Long explanations

Bad:
“ajudamos empresas a crescer”

Good:
“transformamos investimento em leads previsíveis”

---

# Visual Hierarchy

- H1 must dominate visually
- Clear spacing between elements
- Use contrast (size, color, weight)
- Eye should flow naturally:
  Hook → Headline → CTA → Visual

---

# Components

- Rounded cards (12–16px radius)
- Soft shadows
- Metric cards (important for SaaS feel)
- Icon + text blocks
- Testimonials (credibility)
- FAQ (accordion)

---

# Dashboard / Visual System Rules

This is a KEY differentiator.

- Must feel like a real product UI
- Include:
  → Graphs
  → Metrics
  → Labels
- Add depth (layered cards)
- Use subtle gradients inside charts only

Avoid:
- Flat placeholder boxes
- Static-looking visuals

---

# Development Rules

- Clean, structured HTML/CSS
- Lightweight JS only when necessary
- No unnecessary libraries
- Fast loading

- Elementor:
  → clean structure
  → avoid nested chaos

---

# Avoid (STRICT)

- Generic templates
- Washed-out backgrounds
- Flat design with no depth
- Overuse of gradients
- Visual noise
- Weak CTAs
- Static-looking UI

---

# Goal

- Make the website feel like a **growth engine**
- Increase perceived value instantly
- Attract higher-quality leads
- Position SicaCreative as premium and results-driven
